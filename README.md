# Notas del desarrollo del proyecto

## Consideraciones del Plano

El plano se encuentra realizado en draw.io, de forma que si se desea modificar, se puede abrir el archivo `Plano_V3.drawio` en su página web y editarlo a gusto. En el pdf se encuentra la ultima versióon realizada.

## Consideraciones para la implementación de la topologia en Cisco Packet Tracer

Por el momento solo que la topología a partir de los routers es una estrella jerárquica, es decir, cada router se conecta a un router central, y no hay conexiones entre los routers secundarios. Donde el router de la planta baja es le central y los demás se conectan a el.
